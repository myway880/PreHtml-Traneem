1.في الصليب يا الهي
قد رايت حبك
( تفتدي كل اثيم
بدماء ابنك ) 2

2.في الصليب يا الهي
طهرك بدا جلي
( اذ به دنت الخطايا
ايها الرب العلي ) 2

3.في الصليب يا الهي
اشهد العدل الدقيق
( فيه قد جمعت بين
الطهر والحب العميق ) 2

4.في الصليب يا الهي
صلب العالم لي
( وانا صلبت عنه
فيه فخري يعتلي ) 2

5.في الصليب يا الهي
حافز لي في الحياه
( سانادي بفداه
ناشرا دوما لواه ) 2
